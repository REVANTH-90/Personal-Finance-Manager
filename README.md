# 💰 Personal Finance Manager (Java)

## 📌 Overview

The **Personal Finance Manager** is a console-based Java application that helps users manage their daily finances efficiently. It allows users to track income, expenses, and view their current balance using core Java concepts and Object-Oriented Programming.

---

## 🚀 Features

* ➕ Add Income
* ➖ Add Expense
* 📊 View Current Balance
* 📁 Save Transactions to File
* 🗂️ Category-wise Expense Tracking
* 📜 Display All Transactions

---

## 🛠️ Technologies Used

* Java (Core Java)
* OOP Concepts (Inheritance, Polymorphism, Encapsulation)
* Collections Framework (ArrayList, HashMap)
* File Handling (FileWriter)

---

## 🧱 Project Structure

```
├── Transaction.java   (Base class)
├── Income.java        (Child class for income)
├── Expense.java       (Child class for expense)
├── FinanceManager.java (Business logic)
├── FileManager.java   (File handling)
├── Main.java          (User interface)
```

---

## ⚙️ How It Works

1. User selects options from menu
2. Adds income or expense
3. Data is stored in ArrayList
4. Expenses categorized using HashMap
5. Balance calculated using total income - total expense
6. Data can be saved to a file
